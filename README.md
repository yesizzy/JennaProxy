# JennaProxy 
# JennaProxy is not affiliated with Epic Games, and it is not intended to provide unfair advantages, cheating capabilities, or to bypass security systems.

Just a simple, open-source proxy to let you use every cosmetic in the game.

## Why use this?
* **Everything Unlocked:** If it's in the game files, you can wear it.
* **Actually Open Source:** You can read every line of code. No hidden miners, no backdoors.
* **Lightweight:** It won't tank your FPS. It sits in the background and does its job.

## How to get it running
1. Download [Python](http://python.org/downloads/) and install it.
2. Open **start.bat**.
4. That's it! (Press CTRL+C to exit without problems).
